# BrFF APP
NFL Fantasy app (by https://brffootball.com.br/) with customized podcast

## Technology
- React Native
- [Sleeper API](https://docs.sleeper.app/)
- [react-native-track-player](https://react-native-track-player.js.org/docs/basics/getting-started)
- [react-native-image-header-scroll-view](https://github.com/bamlab/react-native-image-header-scroll-view)
- [react-native-navigation-bar-color](https://github.com/thebylito/react-native-navigation-bar-color)
- [react-native-rss-parser](https://www.npmjs.com/package/react-native-rss-parser)
- [react-native-skeleton-placeholder](https://github.com/chramos/react-native-skeleton-placeholder)
- [react-native-walkthrough-tooltip](https://github.com/jasongaare/react-native-walkthrough-tooltip)
- [react-native-text-ticker](https://github.com/deanhet/react-native-text-ticker)
